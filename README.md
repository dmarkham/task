# How to use the web service:
## Clone the repository:
```
% git clone https://github.com/camsn0w/task
```
## Build the program:
```
% cd task
% go build .
```
# Running:
```
% curl "http://localhost:5000"
- or -
visit http://localhost:5000 in a web browser
```
